FrontierQuant — 美股資產配置效率前緣分析儀表板
一個純前端（靜態 HTML/JavaScript）的投資組合最佳化工具，透過蒙地卡羅模擬（Monte Carlo Simulation）與馬可維茲現代投資組合理論（MPT），協助使用者尋找風險與報酬的最佳平衡點（效率前緣），並可直接從 Excel 匯入真實歷史股價資料來計算報酬率、波動度與相關係數。
---
📁 專案結構
```
.
├── index.html        # 主頁面（UI 結構、Tailwind 樣式、各區塊版面）
├── app.js             # 核心邏輯（模擬引擎、Excel 解析、圖表渲染、儲存機制）
├── style.css          # 自訂樣式（index.html 引用，需自行建立／補上）
└── 抓股價.xlsm        # 共用 VBA 工具，用於從 Yahoo Finance 抓取股價並輸出 Excel 報表
```
> ⚠️ `index.html` 有引用 `style.css`，但目前專案內尚未包含此檔案，使用時請自行建立（可為空白檔）或補上自訂樣式。

✨ 主要功能
1. 投資組合參數管理
手動新增／刪除股票資產，設定預期年報酬率與年化波動度
新增資產時，預設與其他股票相關係數為 `0.35`（建議改用 Excel 匯入取得真實相關係數）
2. Excel 匯入歷史股價
支援 `.xlsx` 與 `.xlsm` 檔案格式（透過 SheetJS 解析，依檔案內容判斷格式而非副檔名）
自動偵測兩種資料表格式：
Format A（寬版並排格式）：對應 `抓股價.xlsm` 的輸出，多檔股票在同一張工作表中以「股票代號」標籤橫向並排，每組 3 欄（日期欄、收盤價欄、空白分隔欄），資料從第 4 列（index 3）開始
Format B（舊版逐股工作表格式）：每張工作表代表一檔股票，第 0 欄為日期、第 1 欄為收盤價
匯入時自動略過非資料工作表（如 `BUTTON`、`按鈕`、`SETTINGS`、`設定`）
匯入成功後依日期區間自動計算：
年化預期報酬率（以日對數報酬率年化，乘以 252 個交易日）
年化波動度
各股票間的 Pearson 相關係數矩陣
可調整「歷史數據日期範圍」即時重新計算上述統計量
3. 蒙地卡羅模擬引擎
使用內嵌 Web Worker（Blob URL，相容 `file://` 協定本地開啟）執行運算，避免阻塞主執行緒
可選擇模擬次數：1,000（極速）／2,500（推薦）／5,000（精準）次
隨機生成投資組合權重，計算每組合的：
預期年報酬率
年化波動度（依共變異數矩陣）
夏普比率（Sharpe Ratio，可調整無風險利率 Rf）
自動標記出：
🌟 最大夏普值組合（Max Sharpe）
🟢 全域最小波動組合（GMV, Global Minimum Variance）
4. 視覺化呈現
效率前緣散點圖：X 軸為風險（波動度）、Y 軸為報酬率，標示模擬散點與兩大最佳組合
權重配置圓餅圖：可切換顯示最大夏普或最小波動組合的資產配置比例與金額
資產相關係數熱力圖：紅／白／藍色階呈現正負相關程度
配置比例與績效對比表格：列出兩大最佳組合的報酬率、波動度、夏普比率及各股票配置金額
5. 資料持久化（localStorage）
點擊「儲存資料」可將目前的資產清單、相關係數矩陣、原始股價資料、投資金額、無風險利率、模擬次數與日期區間，存入瀏覽器 `localStorage`（鍵值：`frontierquant_saved_state_v1`）
重新開啟頁面時自動讀取並還原上次儲存的狀態
「重設初始資產」按鈕可清空已匯入的股價資料，回復成預設的 5 檔股票範例（AAPL、NVDA、META、AMZN、TSLA）
6. MPT 科普教學區
頁面下方附有現代投資組合理論的白話文教學，以「雞蛋與籃子」「高速公路比喻」「CP值指標」三個比喻說明分散風險、效率前緣與夏普比率的核心概念。

🔧 抓股價.xlsm（資料來源工具）
一個 Excel VBA 工具，用於批次抓取美股／台股歷史股價，輸出符合本儀表板「Format A」匯入格式的報表。
透過 `MSXML2.XMLHTTP` 呼叫 Yahoo Finance 非官方 Chart API：
`https://query1.finance.yahoo.com/v8/finance/chart/`
JSON 解析以純字串處理完成（無外部函式庫依賴），並使用 `Replace()` 移除內嵌換行字元以避免數值轉換錯誤
台股代號需自行加上後綴：
上市股票：`.TW`
上櫃股票：`.TWO`
輸出工作表第 0 列為「股票代號」標籤列、第 1 列為欄位標題（日期／收盤價）、第 2 列空白、第 3 列起為實際資料
> ⚠️ Yahoo Finance Chart API 為非官方介面，回應格式可能隨時變動，若匯入失敗請先確認該 API 是否仍可正常存取。
---
🚀 使用方式
直接用瀏覽器開啟 `index.html`（支援 `file://` 本地開啟，無需架設伺服器）
（可選）使用 `抓股價.xlsm` 抓取最新股價，產出 `.xlsx`／`.xlsm` 報表
點擊「從 Excel 匯入」上傳報表，系統自動計算各股票統計量與相關係數
調整投資總額、無風險利率、模擬次數等參數
點擊「開始計算」執行蒙地卡羅模擬
查看效率前緣散點圖、最佳配置權重與績效對比表格
點擊「儲存資料」保存當前設定，下次開啟自動還原

⚠️ 已知限制與注意事項
Excel 匯入需至少 2 檔有效股票資料，否則會跳出「至少需要 2 個有效工作表」相關錯誤提示，請確認檔案中每檔股票皆有完整日期與收盤價欄位
手動新增股票時，與既有股票的相關係數固定為 `0.35`，僅供粗略估算，建議優先以 Excel 匯入取得真實歷史相關係數
`localStorage` 資料僅存於目前瀏覽器、目前裝置，清除瀏覽器快取或更換裝置將遺失已儲存的設定
本系統之模擬結果僅供學術研究與資產配置科普教學參考，不構成任何實際市場交易投資之依據
